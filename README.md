# Pousada Me Leva Logo

Projeto Next.js pronto para seguir com a landing page da pousada.

## Rodar localmente

```bash
npm install
npm run dev
```

## O que já tem

- Página única pronta
- SEO básico e Open Graph
- JSON-LD para Google
- CTA fixo para WhatsApp
- Layout responsivo
- Espaços para trocar imagens e textos depois

## Onde editar

- `app/page.tsx` → conteúdo da landing
- `app/layout.tsx` → meta tags globais
- `app/globals.css` → estilos
